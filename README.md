# 小猫咪邮件在线发送

#### 介绍
一款免登录发送邮件，后台可添加邮箱,前台可选择发送邮箱，无需数据库

#### 软件架构
网站数据采取本地保存，所以，使用前请给网站修改权限，否则很多功能将无法使用


#### 安装教程

1. 上传服务器或者主机
2. 登录后台，添加发送邮箱即可

#### 使用说明

1.  强烈建议修改后台地址
2.  后台地址：域名/admin.php 默认账号密码为admin/123456
3.  网站数据采取本地保存，所以，使用前请给网站修改权限，否则很多功能将无法使用
